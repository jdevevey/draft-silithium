# Silithium Internet-Draft

This is the working area for the CFRG individual Internet-Draft, "Silithium - A Compact, Efficient and Non-separable Hybrid Signature".

## Contributing

We welcome contributions via pull requests, github issues, or emails to the authors.

## Command Line Usage

Formatted text and HTML versions of the draft can be built using `make`.

```sh
$ make
```

Command line usage requires that you have the necessary software installed.  See
[the instructions](https://github.com/martinthomson/i-d-template/blob/main/doc/SETUP.md).
